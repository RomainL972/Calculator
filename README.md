# Calculator

This is an arbitrary-precision calculator

## Building
This project uses cmake
```
mkdir build
cd build
cmake ..
make -j(number of cores+1)
```

## Testing
I use my own php script for testing, tests.php
```
../tests.php
```

## Running
```
./Calculator 0123456789
1+1
```
